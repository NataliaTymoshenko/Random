🎲 Імітація кубика

Киньте віртуальний кубик і подивіться, як це реалізувати в Python!

🎲
import random # Кидок кубика (числа від 1 до 6) dice = random.randint(1, 6) print(f"Випало число: {dice}") # Або використовуємо діапазон з кроком (необов'язково) # randrange(1, 7) — від 1 до 6 (7 не включно!) dice2 = random.randrange(1, 7)
💡 Важливо:
randint(1, 6) — включає ОБИДВА кінці (1 і 6)
randrange(1, 7) — включає початок, але НЕ включає кінець!

🛠️ Інші корисні методи

🎯 Випадковий вибір

import random fruits = ["🍎", "🍌", "🍇", "🍊"] winner = random.choice(fruits) print(winner) # Випадковий фрукт

choice() — вибирає один елемент із списку

🔢 Випадковий дріб

import random # Випадкове число від 10.5 до 20.5 price = random.uniform(10.5, 20.5) print(f"Ціна: {price:.2f} грн")

uniform(a, b) — випадкове ДРОБОВЕ число в діапазоні

🔄 Перемішування

import random cards = ["Туз", "Король", "Дама", "Валет"] random.shuffle(cards) # Перемішує список! print(cards)

shuffle() — перемішує список "на місці" (змінює оригінал)

📊 randrange()

import random # Парні числа від 0 до 100 even = random.randrange(0, 101, 2) print(even) # 0, 2, 4, ..., 100 # Кратні 5 five = random.randrange(0, 51, 5)

randrange(start, stop, step) — з кроком

📚 Основні методи модуля random

Метод Що робить Приклад
random() Число від 0.0 до 1.0 0.374448871756
randint(a, b) Ціле число [a, b] randint(1, 6) → 4
randrange(a, b, step) З кроком, b не включно randrange(0, 10, 2) → 4
uniform(a, b) Дробове число [a, b] uniform(1.5, 2.5) → 1.89
choice(seq) Вибір зі списку choice(['a', 'b']) → 'a'
shuffle(list) Перемішує список [3, 1, 2]
sample(population, k) k унікальних елементів sample(range(10), 3)
⚠️ Важливо пам'ятати:
  • Спочатку завжди пишемо: import random
  • randint() включає обидва кінці діапазону
  • randrange() НЕ включає правий кінець (як range())
  • shuffle() змінює список "на місці" і не повертає новий список
  • Для криптографії (паролі, ключі) використовуйте secrets замість random!

Задача 1: Генератор паролів (простий)

Створіть програму, яка генерує 4-значний числовий PIN-код.

import random # Генеруємо 4-значний PIN (від 1000 до 9999) pin = random.randint(1000, 9999) print(f"Ваш PIN-код: {pin}") # Або по одній цифрі (для розуміння) pin2 = "" for i in range(4): digit = random.randint(0, 9) pin2 += str(digit) print(f"PIN: {pin2}")
Пояснення: Використовуємо randint(1000, 9999) для гарантії 4 цифр, або цикл для покрокової генерації з конкатенацією рядків.

Задача 2: Кидок двох кубиків

Змоделюйте кидок двох гральних кубиків і виведіть їх суму.

import random # Кидаємо два кубики dice1 = random.randint(1, 6) dice2 = random.randint(1, 6) total = dice1 + dice2 print(f"Кубик 1: {dice1}") print(f"Кубик 2: {dice2}") print(f"Сума: {total}") # Перевірка на "подвійне" if dice1 == dice2: print("🎉 Подвійне! Бонусний хід!")
Пояснення: Генеруємо два незалежних числа від 1 до 6. Кожен виклик randint() — це окремий "кидок".

Задача 3: Випадковий колір

Оберіть випадковий колір із списку: червоний, зелений, синій, жовтий, фіолетовий.

import random colors = ["червоний", "зелений", "синій", "жовтий", "фіолетовий"] # Випадковий вибір lucky_color = random.choice(colors) print(f"Сьогоднішній щасливий колір: {lucky_color}") # Вибір кількох кольорів без повторень palette = random.sample(colors, 3) print(f"Ваша палітра: {palette}")
Пояснення: choice() — один елемент, sample() — кілька унікальних елементів (без повторень).

Задача 4: Гра "Вгадай число"

Комп'ютер загадав число від 1 до 100. Перевірте, чи вгадала користувач (імітація).

import random # Комп'ютер загадав число secret = random.randint(1, 100) # Імітація вгадування (в реальності — input()) guess = random.randint(1, 100) # Імітуємо введення print(f"Ви ввели: {guess}") if guess == secret: print("🎉 Вітаємо! Ви вгадали!") elif guess < secret: print("Загадане число більше") else: print("Загадане число менше") print(f"Загадане число було: {secret}")
Пояснення: Генеруємо секретне число один раз, потім порівнюємо з вгадуванням. Використовуємо умовні оператори для підказок.

Задача 5: Перемішування колоди

Створіть колоду карт (4 масті по 9 карт) і перемішайте її.

import random # Створюємо колоду suits = ["♠", "♥", "♦", "♣"] ranks = ["6", "7", "8", "9", "10", "В", "Д", "К", "Т"] # Генеруємо всі комбінації deck = [] for suit in suits: for rank in ranks: deck.append(rank + suit) print(f"Колода до перемішування: {deck[:5]}...") # Перемішуємо random.shuffle(deck) print(f"Після перемішування: {deck[:5]}...") print(f"Верхня карта: {deck[0]}")
Пояснення: Вкладені цикли створюють колоду. shuffle() змінює порядок елементів у списку випадковим чином.
